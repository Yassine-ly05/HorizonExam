# HorizonExam &mdash; Login Page (Front-End Only)

This project contains a **single-page React front-end** implementing the HorizonExam login screen for the Horizon School of Digital Technologies.

## Stack

- React 18
- Vite (development/build tooling)
- Plain CSS (no UI framework)

## Development

From the project root:

```bash
npm install
npm run dev
```

Then open the printed local URL (by default `http://localhost:5173`) in a browser.
